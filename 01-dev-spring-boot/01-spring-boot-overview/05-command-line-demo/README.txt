1. I NEED TO ADD THE DEPENDENCY:

    <dependency>
        <groupId>org.springframework.boot</groupId>
        <artifactId>spring-boot-starter-actuator</artifactId>
    </dependency>


2. TO EXECUTE FROM COMMAND LINE, I HAVE TO USE THE COMMAND "mvn package" in the folder where i got my project
3. EXECUTE THIS COMMAND "java -jar target\projectName-0.0.1-SNAPSHOT.jar"
4. TO STOP APP USE "CTRL + C"


mvn spring-boot:run
